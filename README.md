# 📦 光影纸盒图纸生成器 (Shadow Box Template Generator)

> 💡 **“用代码雕刻光影，将平面图像升维成现实世界的光学魔法。”**

这是一个纯前端的交互式光学装置设计工具。用户只需上传一张普通的 2D 图像，并输入想要制作的“实体纸盒”尺寸，系统便会利用**逆向光线追踪算法**，自动计算出光线穿过纸盒侧壁时的空间交点，最终生成可用于 1:1 真实打印、手工裁切的图纸（PDF格式）。

## ✨ 核心功能特性

* 🖼️ **自定义图像光栅化**：支持任意图片拖拽上传，内置阈值调节与黑白反转，实时提取光影剪影。
* 📐 **全动态物理空间控制**：支持精确到 0.1cm 的滑块与数值双向输入。可动态调节纸盒长宽高、灯泡高度，甚至**灯泡在平面内的 X/Y 轴偏移量**。
* 👁️ **WebGL 实时光学可视化**：基于 `Three.js` 构建 3D 空间，实时渲染半透明雕刻纸盒、发光灯泡，以及真实的落地投影和光路射线。
* 🛡️ **防呆物理连通性检测**：内置基于 BFS（广度优先搜索）的“悬空孤岛”检测算法。若图案在剪裁后会完全掉落，系统会在图纸上以**红色高亮警示**。
* 🖨️ **工业级 1:1 打印输出**：一键导出包含 6 页的 A4 PDF 打印册（配置说明 + 底面贴纸 + 4面侧壁切图），严格按物理 1:1 比例生成，告别尺寸误差。
* ⚡ **开箱即用 (Zero-Config)**：100% 纯静态前端应用，无需后端服务器，打开网页即可使用。

## 🧠 硬核算法原理

1. **逆向空间光线追踪 (Reverse Raycasting)**：系统以最终目标投影的每一个像素点为起点，向虚拟点光源发射射线，求出射线与纸盒四壁边界的 3D 交点，从而决定纸盒上对应位置是应该“掏空”还是“保留”。
2. **悬空碎片排查 (Island Detection)**：将侧壁生成的黑白切图视为连通图，从边界安全区向内进行广度优先搜索（BFS）。未被遍历到的实体区域即为物理上无法实现的“悬空碎片”，系统将其标记为红色。

## 🛠️ 技术栈

* **核心逻辑 & 界面**：原生 HTML5 / CSS3 / JavaScript (ES6+)
* **3D 引擎**：[Three.js](https://threejs.org/) (结合 OrbitControls 实现交互式视角)
* **PDF 生成**：[jsPDF](https://parall.ax/products/jspdf) (处理 2D Canvas 到 PDF 的无损物理映射)

## 💻 本地运行与部署

### 本地运行

本项目无任何环境依赖。您可以直接在浏览器中双击打开 `index.html`。
*(注：为避免浏览器的本地跨域 `CORS` 限制导致无法读取本地图片，建议使用 VS Code 的 **Live Server** 插件，或通过 Python 开启本地服务：`python -m http.server 8000`)*

### 免费部署到 GitHub Pages

1. Fork 或克隆本仓库到您的 GitHub。
2. 进入仓库的 **Settings** > **Pages**。
3. 在 **Build and deployment** 下的 Source 选择 `Deploy from a branch`。
4. Branch 选择 `main` 分支和 `/(root)` 文件夹，点击 **Save**。
5. 等待 1-2 分钟，您的专属工具网站即可在全球上线！

## ✂️ 实际制作与排坑指南

1. **准备材料**：推荐使用具有一定厚度且不透光的硬纸板（如快递盒）或深色亚克力板。
2. **打印图纸**：在打印导出的 PDF 时，**务必在打印机设置中选择“实际大小”或“100% 缩放”**，切勿选择“适应纸张”，否则尺寸将无法与您的物理盒子对齐。
3. **红色警告处理**：如果在侧壁图纸上看到红色区域，说明这块纸板在掏空周围后会掉落。**解决方法**：在实际动刀时，手工保留 1-2 毫米的纸条作为“桥梁”使其相连，或者在剪裁后用细铁丝/透明胶带将其固定在原位。
4. **光源选择**：尽量使用**点光源**（如手机闪光灯、小灯珠），发光面积越小，投影的边缘越锐利。
